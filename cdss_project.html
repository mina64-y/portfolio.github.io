프로젝트 시연

영상 소개

CDSS 시스템의 전체적인 기능과 사용자 인터페이스를 확인할 수 있는 데모 영상입니다.

주요 내용

• 역할별 로그인 시스템
• AI 기반 진단 보조 기능
• 환자 데이터 통합 관리
• 모바일 앱 연동 서비스

시스템 아키텍처

기능 소개

의사 대시보드

소개

간호부, 영상의학과, 원무과에서 입력된 임상 데이터, 오믹스 데이터, 영상 데이터를 통합적으로 수집하고 대시보드에서 환자의 상태를 한눈에 파악할 수 있도록 구성했습니다.

기능

환자 확인, 간호 일지 확인, CT분석 결과 확인, 임상 데이터 예측, 약물 상호작용, AI논문 검색, 환자 예약 캘린더

간호부

기능

간호일지 AI 자동작성, OMICS 입력 및 분석, 클리닉 데이터 입력 및 분석

영상의학과

기능

CT 업로드, CT확인, AI 영상 분석, 3D 이미지 생성 및 시각화

원무과

기능

환자 접수, 예약 관리, 앱 인증코드 발급, 환자 정보 관리

환자 맞춤 모바일 서비스

기능

실시간 예약, 예약확인, 병원, 응급실, 약국 검색
